# Archived Components

These components are archived for reference but not part of the core application.

## Contents

### notebooks/
Jupyter notebooks for data analysis and exploration:
- `cohort_retention_analysis.ipynb` - Customer retention cohort analysis
- `customer_segmentation.ipynb` - ML-based customer segmentation
- `funnel_ab_test_analysis.ipynb` - A/B test funnel analysis
- `revenue_forecasting.ipynb` - Revenue prediction models

### generators/
Synthetic data generation tools:
- `synthetic_data.py` - Generate fake data for testing
- `cli.py` - Command-line interface for data generation

### benchmarks/
Performance benchmarking scripts:
- `query_benchmarks.py` - Database query performance tests

---

To restore any component, move it back to the project root:
```bash
mv _archive/notebooks ../notebooks
```
